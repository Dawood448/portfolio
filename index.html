SPORTS MANAGEMENT PLATFORM for COLLEGE

It is a flutter based application “SMPC”. It's aim is to provide a detail environment to the sports management authority so that these activities should be managed effectively. Its an application in come with multiple functions that provides facilities about player registration, teams management,add multiple sports and the most important feature create and manage events, also add events realated pictures as a memory etc. In event section admin create an event set time, date, venue and price money. The other college admin has option to see the event details and accept the challenge for a sports related match. These events and gallery view also available offline so that if you are run out of internet,the created and happening events will show on home page.
